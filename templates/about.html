About StereoType

StereoType is a Software Engineering class project aimed at providing music recommendations based on user preferences. Leveraging the Spotify API, our application analyzes user data to generate personalized recommendations. Developed using a combination of HTML, CSS, JS, and Python, StereoType showcases the principles of web development and API integration. Through this project, we gained hands-on experience in building a responsive and user-friendly web application. We're proud to present StereoType as a functional and innovative solution for music enthusiasts.
